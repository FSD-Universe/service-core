# ServiceCore

![ProjectLanguageCard]![ProjectLicense]![BuildStateCard]

## 介绍

ServiceCore 是一个基于 Go 语言的框架，提供了服务框架，用于开发分布式服务。

## 功能

- 日志记录
- 配置管理
- 缓存操作
- Http响应封装

## 使用

```shell
go get half-nothing.cn/service-core
```

## 测试

```shell
go test ./...
```
